# Temporary Images Directory

This directory is used to store the daily gig guide images that are generated by the GitHub Actions workflow.

## Process

1. Images are generated daily at 7:35 AM Melbourne time
2. Generated images are stored here temporarily
3. After successful posting to Instagram, images are automatically cleaned up

## Image Naming Convention

Images follow this naming pattern:
- `gigs_YYYYMMDD_carousel0.png` - Title slide
- `gigs_YYYYMMDD_carousel1.png` - First gig slide
- `gigs_YYYYMMDD_carousel2.png` - Second gig slide
- etc.

Where YYYYMMDD is the date in format like 20240213

## Manual Review

You can review the generated images here before they are posted to Instagram. The posting process is separate and must be manually triggered through the GitHub Actions interface.

## Cleanup

Images are automatically cleaned up after successful posting to Instagram, but this README.md file is preserved.
